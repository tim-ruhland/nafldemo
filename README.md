# NAFL Football Demo

Play a game of NAFL football vs. computer

https://tim-ruhland.github.io/nafldemo/

Team 1 (T1) - You

Team 2 (T2) - Computer player

15 plays per quarter

T1 kicks off to start game

T2 kicks off at start of 2nd half

Offense options:
- Running plays - Dive, End Run, QB Sneak
- Passing plays - Short Pass, Bomb
- Trick Play - only one per game
- Special teams - Punt, Field Goal

Defense options:
- Run 
- Pass
- Normal

The Field Goal button will show the percentage chance of making it from the spot.

Calling the correct Run/Pass defense will result in increased turnover chance and reduced big play chance.

Calling the incorrect Run/Pass defense will result in no chance of turnover and increased chance of big play.
